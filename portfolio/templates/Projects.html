Auction Bidding Platform using Full Stack

The Auction Bidding Platform is a robust and feature-rich web application designed to streamline the process of online auctions. Developed using Node.js and Express.js, it offers a seamless experience for hosting and participating in auctions. The platform features real-time bidding, allowing users to place and track bids in real-time. It includes secure user authentication and an intuitive interface for browsing, listing, and managing auction items. A key highlight of the platform is its integration with Stripe for secure and efficient payment processing. This ensures that buyers can complete transactions seamlessly while sellers receive payments promptly and reliably. Designed for scalability and performance, the platform is equipped to handle a diverse range of auction requirements. Whether for small-scale events or large marketplaces, this solution delivers a smooth, safe, and engaging user experience.

Screen Recording Application Using Python

The Screen Recording Application is a lightweight and efficient tool developed using Python to record on-screen activities seamlessly. Built with simplicity and functionality in mind, this application enables users to capture their screens effortlessly, making it perfect for creating tutorials, recording gameplay, or saving virtual meetings. The project leverages popular Python libraries to ensure high-quality recordings with minimal system resource usage. Users can customize their recording preferences, including frame rates and output formats, to suit their needs. With clean, modular code, this application is also an excellent example of Python's versatility in developing desktop utilities.
